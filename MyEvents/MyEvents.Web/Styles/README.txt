To have some intellisense on .less files, go to Tools-> Options-> Text Editor-> File Extension 
and associate the .less extension to CSS Source Editor or use a visual studio extension like http://visualstudiogallery.msdn.microsoft.com/07d54d12-7133-4e15-becb-6f451ea3bea6

To know more about LESS language go to http://lesscss.org/
